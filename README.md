    <summary>11. graph</summary>
    그래프 자료구조를 인접리스트를 활용하여 구현, 그래프 탐색 알고리즘과 그에 대한 응용문제 풀이

        - 예제
        1. DFS
            깊이 우선 탐색을 직접 구현함
            재귀를 활용한 방법, Stack를 활용한 방법으로 나뉘어져 있으며, 재귀를 활용해 간단하게 문제를 풀이함
            탐색, 최단경로가 아닌 대부분의 문제는 깊이 우선 탐색을 고려

        2. BFS
            너비 우선 탐색을 직접 구현함
            QUEUE를 활용한 방법으로 구현 했으며, 최단경로 문제에서 특히 많이 활용

        3. Dijkstra
            다익스트라 알고리즘을 직접 구현함
            가중치가 있는 그래프에서 최단 경로를 BFS 알고리즘을 통해 보장함
            *** 음의 가중치가 있을 경우 다익스트라 알고리즘은 최적해를 보장하지 않는다. 이 점을 유의할 것 ***
            Node라는 클래스를 도입해 dest, cost를 관리 함으로서 최단경로를 구하려 했음
            *우선순위 큐*를 도입하여 가중치를 오름차순으로 정렬 후 탐색을 이어 감

        - 응용
        4. Escape *<중요>*
            좌표계에서 최단거리를 보장하는 알고리즘을 구현 해야 함
            이차원 배열을 통해 좌표계를 구현 후
            Node를 통해 좌표 위치를 특정, rx ry를 통해 Node의 움직임을 간소화 시킴
            반복문을 통해 rx ry로 Node 움직임을 특정하고 맵 밖이나 벽으로 갈 경우 예외처리를 시킴
            최단거리는 BFS 알고리즘을 활용해 문제를 풀이 함
            좌표계를 다루는 기본적인 문제이므로 자주 풀이 할 것
